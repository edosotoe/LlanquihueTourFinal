# Actividad Sumativa 1: Realizando validación, excepciones y relaciones entre clases

## Llanquihue Tour - Sistema de Gestión

## Descripción
Este proyecto modela a las personas vinculadas a la agencia de turismo "Llanquihue Tour" 
mediante clases organizadas en paquetes, y aplica principios de encapsulamiento, 
composición y herencia.

## Estructura del Proyecto
```
src/
|__ app/
|   |__ Main.java
|__ model/
|__ Persona.java
|__ Direccion.java
|__ Cliente.java
```

## Instrucciones para compilar y ejecutar

1. Abrir el proyecto en el IDE (NetBeans)
2. Verificar que la clase principal sea `app.Main`
3. Ejecutar con el botón "Run"
